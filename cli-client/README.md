# Peel Console Client

Interactive console UI for exploring Bitcoin offers via the Peel API.

## Setup

1. Install dependencies:
```bash
cd cli-client
pip install -r requirements.txt
```

## Running

From the `cli-client` directory:

```bash
python src/main.py
```

## Features

Currently implemented:
- Interactive menu navigation
- Offers menu (stub)
- Settings menu (configuration display)

Coming soon:
- Search offers with filtering
- Market volatility analysis
- Real API integration
